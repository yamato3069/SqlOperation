プロジェクト名（仮）

ローカルLLMを用いた「自然文 → SQL → MySQL参照」システム（Go + React）

1. 概要

本システムは、ユーザーが自然文で質問するとローカルLLMがSQLを生成し、MySQLに問い合わせて結果を表示する仕組み。
SQLを直接書かずにデータベース参照を行える体験を目的に、Go（バックエンド）とReact（フロントエンド）で実装。

2. 背景・目的（自己学習の狙い）

生成AI（LLM）を業務でどう活用できるか検討したかった

SQLに不慣れでもDB参照ができると作業が楽になるのでは、と考えた

Go と React の学習（API/画面分離、実装の組み立て方の理解）を目的とした

3. システム概要（処理の流れ）

ユーザーが自然文で質問する

ローカルLLMが質問文をもとにSQLを作成する

生成されたSQLでMySQLへ問い合わせる

取得結果を画面に表示する

4. 使用技術

フロントエンド：React

バックエンド：Go

DB：MySQL

LLM：ローカル実行（外部へ送信しない前提）

5. 仕様・制約
5.1 読み取り専用

DBに変更を加える処理は行わない（例：DELETE / UPDATE / INSERT は不可）

本システムは閲覧用途のみ

5.2 活用シーンが未確定（現時点の課題）

「DBを閲覧するだけ」であれば既存のクライアントツールで十分なケースも多く、
本システムをわざわざ使う明確な場面がまだ整理できていない
